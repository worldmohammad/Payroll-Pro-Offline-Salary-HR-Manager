# 💰 Payroll Pro — Offline Salary & HR Manager

**100% Client-Side. Zero Server. Absolute Privacy.**

A comprehensive offline payroll management system. Manage departments, employees, calculate salaries with tax/insurance deductions, generate professional payslips, and track payroll history—all without an internet connection.

## ✨ Features

- **🏢 Department Management:** Create and manage organizational departments
- **👥 Employee Directory:** Add employees with base salary and department assignment
- **🧮 Payroll Calculator:** Auto-calculate gross pay, tax, insurance, and net salary
- **📄 Professional Payslips:** Generate detailed payslips with earnings and deductions breakdown
- **📊 Dashboard:** Real-time stats, department cost chart
- **📋 Payroll History:** Track all generated payslips with search and filters
- **🖨️ Print & PDF:** Export payslips (Pro)
- **💾 Backup/Restore:** JSON export/import (Pro)
- **📥 CSV Export:** Download payroll history (Pro)
- **🔒 100% Offline:** All data in IndexedDB
